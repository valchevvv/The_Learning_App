This is the back-end of this app.

To run it you will need to do the following steps:
1. If there is no .env file you will need to create it or default `DATABASE_URL` - mongodb connection string will be your localhosted mongodb and default running port will be 1337.
2. Run `npm i` to install all modules that are required
3. You need to put your mongodb connection string to `DATABASE_URL` in `.env` file.
4. You can change running port if you want. You can do this from `PORT` in `.env` file.

5. To run it just type `npm run start`.
